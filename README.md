# 冰-金属摩擦物理实验平台

这是一个面向物理竞赛实验《冰与金属的摩擦系数测量》的可视化平台，采用蓝白简约科技风设计。

## 平台模块

- 仿真模拟平台：设置金属试样、电机转速、温度、载荷和信号放大倍率，动态模拟旋转圆盘法的冰-金属滑动摩擦测量过程。
- 实时检测平台：实时显示空载电压、净摩擦电压、摩擦力矩和滑动摩擦系数。
- 数据输出平台：展示银、铜、铁在 0/400/800/1200/1600/2000 rpm 电机转速下的数据表，并支持记录当前工况。
- 数据导出平台：支持导出 CSV、JSON 和生成实验摘要。

## 实验依据

平台依据最新竞赛报告中的旋转圆盘法建模：

- 材料：银、铜、铁三种金属圆柱。
- 工况：0 ℃恒温冰面，电机转速 0/400/800/1200/1600/2000 rpm。
- 减速：20:1 高精度行星减速器，圆盘实际转速为 0-100 rpm。
- 采集链路：滑动摩擦产生阻力矩，扭矩传感器转化为电压，经放大后计算净摩擦电压。
- 求解逻辑：由旋转圆盘法直接积分求解，滑动摩擦系数公式为 `μs = 3T / (2NR)`。
